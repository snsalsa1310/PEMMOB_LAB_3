# Project Description

Anime Verse merupakan aplikasi mobile yang dibangun menggunakan Flutter, yang berfungsi sebagai
platform untuk menampilkan berbagai jenis anime. Pengguna bisa melihat daftar anime lengkap dengan
judul, genre, gambar, dan deskripsi singkat dari setiap anime.
